# Rust Music

`rust-music` is a framework for programmatic music manipulation and composition.

## Features

None (for now)

## Goals

* Provide a complete and easy-to-use library to compose and generate music
* Export to ABC files
* Import ABC files
* Export to MIDI files
* Import from MIDI files
* Be rust-idiomatic
